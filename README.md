## 🌟 About This Project

This is my **first ever Python project**, built from scratch as part of my
self-learning journey into programming and fintech.

I started with zero coding knowledge and used this project to learn the
fundamentals of Python and Streamlit — turning a simple terminal-based
script into a fully interactive web application.

---

## 🚀 Features

- 💵 **Income tracking** — Save your monthly income with one click
- 📝 **Expense logging** — Add multiple expenses by name and amount
- 📊 **Visual dashboard** — Pie chart and bar chart breakdowns
- 🏦 **Balance summary** — Real time income vs expenses vs balance
- ⚠️ **Smart warnings** — Alerts when you are overspending
- 💾 **Data persistence** — Expenses saved locally to CSV file
- 🇸🇬 **Singapore focused** — Budgeting tips tailored for Singaporeans

## 📚 What I Learned

### Python Fundamentals
- ✅ **Variables** — Storing and updating data like income and expenses
- ✅ **Lists** — Using `[]` to collect multiple expense entries
- ✅ **Loops** — `while` and `for` loops to iterate through expenses
- ✅ **Conditionals** — `if/elif/else` for budget warnings and validations
- ✅ **Functions** — Writing reusable functions like `save_expenses()`
- ✅ **File handling** — Reading and writing to CSV files
- ✅ **F-strings** — Formatting currency output cleanly
- ✅ **Float & type conversion** — Handling numeric input correctly
- ✅ **Debugging** — Fixing real errors like `NameError` and `SyntaxError`

### Streamlit
- ✅ **st.number_input / st.text_input** — Collecting user input
- ✅ **st.button** — Triggering actions on click
- ✅ **st.session_state** — Persisting data across interactions
- ✅ **st.columns** — Creating side by side layouts
- ✅ **st.metric** — Displaying key financial figures
- ✅ **st.rerun** — Refreshing the app after user actions
- ✅ **st.success / st.warning / st.error** — User feedback messages
- ✅ **st.plotly_chart** — Embedding interactive charts

### Tools & Workflow
- ✅ **VS Code** — Writing and running code
- ✅ **Git & GitHub** — Version control and project showcase
- ✅ **pip** — Installing Python libraries
- ✅ **Terminal / PowerShell** — Running scripts and managing files

## ▶️ How To Run It

Make sure Python and Streamlit are installed:

```
pip install streamlit plotly pandas
```

Then run:

```
streamlit run budget_app.py
```

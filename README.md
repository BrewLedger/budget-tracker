# Budget Tracker 💰

A simple personal budget tracker built with Python.
This is my first ever Python project!

## What it does
- Enter your monthly income
- Add multiple expenses with names and amounts
- See a full summary of your spending
- View your remaining balance

## How to run it
Make sure you have Python installed, then run:
python basic_budget_tracker.py

## Example output
============================
        YOUR SUMMARY        
============================
Income:   $2500.0
Expenses: $1190.0
----------------------------
Balance:  $1310.0
============================
